# PocketScope 系统架构图 —— AI 绘图指令

> 将以下内容发给 ChatGPT（支持 DALL·E / GPT-4o 图像生成），或任何支持文生图的 AI 工具，即可生成系统的架构图和 UI 界面图。

## 🎯 Prompt 1：系统整体架构框图（推荐先画这个）

```
请生成一张专业的技术架构框图（block diagram），展示一个基于 FPGA 的"信号发生器 + 双通道示波器"多功能掌上仪器系统。要求如下：

【整体风格】
- 深色背景（深蓝灰或纯黑），科技感配色（霓虹绿、青色、黄色、蓝色线条和文字）
- 框图使用圆角矩形，不同功能区域使用不同颜色标识
- 所有文字使用英文标注
- 图片比例 16:9，适合 PPT 或文档插图
- 类似 Xilinx/Intel 官方数据手册中的架构图风格

【系统包含以下模块，用箭头标注数据流向】：

1. 顶部中央 — FPGA 芯片（标注 "XC7A35T-1CSG324C"，大矩形框）
   内部子模块用虚线分隔：
   - UI Controller (ui_ctrl) — 按键消抖/模式解码/参数调节
   - DDS Core (dds_core) — 24-bit 相位累加器，正弦/方波/三角波
   - Modulation (modulation) — AM/FM/SPWM 调制
   - DAC Controller (dac0832_ctrl) — DAC0832 时序控制
   - Clock Divider (clk_div_25m) — 100MHz→25MHz
   - VGA Controller (vga_ctrl) — 640×480@60Hz 时序生成
   - Waveform Display — 双通道波形渲染 + 网格 + 字符叠加
   - Lissajous Display — X-Y 模式 + 余辉拖尾
   - Kaleidoscope — 12-fold 对称艺术图案 + 色轮
   - Oscilloscope Acquisition — XADC 双通道连续采样
   - Wave Analyzer ×2 — 频率/幅度/RMS/占空比/上升时间
   - Char Gen — 8×8 ASCII 点阵字体 ROM
   - Wave RAM CH1/CH2 — 双通道波形存储 (1024×8)

2. 左侧输入区：
   - 5 个按键 (BTN0–BTN4)：Center/Up/Down/Left/Right
   - 8 个拨码开关 (SW0–SW7)：模式/波形/频率粗调
   - 8 个 DIP 开关 (DIP0–DIP7)：频率细调
   - 2 个 BNC 模拟输入 (CH1/CH2) → XADC

3. 右侧输出区：
   - VGA 显示器 (640×480@60Hz)
   - DAC0832 → BNC 模拟输出 (正弦/方波/三角波, 100Hz–10kHz)

4. 底部状态栏：
   四大模式指示灯（用文字表示）：
   - 00: Signal Generator | 01: Oscilloscope | 10: Lissajous | 11: Kaleidoscope

5. 数据流箭头标注：
   - 按键/开关 → UI Controller
   - UI Controller → DDS Core (频率字/波形选择)
   - UI Controller → Modulation (调制类型/深度)
   - DDS Core → Modulation → DAC Controller → DAC0832 → BNC
   - XADC → Oscilloscope Acq → Wave RAM → Waveform Display → VGA
   - Wave RAM → Wave Analyzer → 参数反馈到显示
   - 所有显示模块 → Mode MUX → VGA输出

6. 在框图右下角添加小字标注： "PocketScope v2.0 | EGO1 Dev Board | 2026"
```

## 🔧 使用说明

1. **单个生成**：将上述任一 Prompt 单独复制发给 ChatGPT（需开启 DALL·E 或图像生成能力）
2. **批量生成**：将所有 Prompt 依次发送，每个都会生成不同的图
3. **微调**：根据需要修改参数（频率值、波形类型、颜色等）
4. **最佳模型**：推荐使用 GPT-4o（图像生成）或 DALL·E 3
5. **替代工具**：也可以使用 Midjourney、Stable Diffusion，但需将 prompt 翻译为英文并调整格式
