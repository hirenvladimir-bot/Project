# PocketScope 多功能掌上仪器 使用说明书

## 1. 概述

PocketScope 是一款基于 Xilinx XC7A35T (EGO1开发板) 的集成化多功能测量仪器，集**数字示波器**、**信号发生器**、**李萨如图形**和**万花筒显示**于一体，通过 VGA 显示器输出 640×480 实时画面。

---

## 2. 硬件连接

| 接口 | 连接目标 |
|------|----------|
| VGA 接口 | VGA 显示器 (640×480@60Hz) |
| DAC0832 输出 | 示波器 / 外部电路 (BNC) |
| J5 扩展口 pin9-10 | 模拟信号输入 CH1 (XADC VAUXP2/N2) |
| J5 扩展口 | 模拟信号输入 CH2 |
| 电源 | EGO1 USB 供电 (5V) |

---

## 3. 模式选择

使用 **SW3** 和 **SW2** 切换四大工作模式：

| SW3 | SW2 | 模式 | 说明 |
|-----|-----|------|------|
| 0 | 0 | **示波器** (Scope) | 双通道时域波形显示 |
| 0 | 1 | **李萨如** (Lissajous) | X-Y 合成图形 |
| 1 | 0 | **信号发生器** (SigGen) | 输出正弦/方波/三角波 |
| 1 | 1 | **万花筒** (Kaleidoscope) | 8-fold 对称艺术图案 |

---

## 4. 信号发生器模式（SW3=1, SW2=0）

### 4.1 波形选择

| SW1 | SW0 | 波形 | VGA 显示 |
|-----|-----|------|----------|
| 0 | 0 | 正弦波 (Sine) | W:N |
| 0 | 1 | 方波 (Square) | W:S |
| 1 | 0 | 三角波 (Triangle) | W:T |
| 1 | 1 | 正弦波 (默认) | W:N |

### 4.2 频率调节

- **粗调**: SW[7:5] 选择频率量程
  - `000`: 100Hz ~ 5200Hz (步进 20Hz)
  - `001`: 2000Hz ~ 7100Hz
  - `010`: 4000Hz ~ 9100Hz
  - `011`: 6000Hz ~ 11100Hz
  - `100`: 8000Hz ~ 13100Hz
- **细调**: DIP[7:0] 拨码开关，0-255 × 20Hz
- 输出频率自动钳位在 **100Hz ~ 10kHz**

### 4.3 幅度调节

| 按键 | 功能 |
|------|------|
| **BTN Center** (PB0) | 幅度 +10 (最大 255) |
| **BTN Up** (PB1) | 幅度 -10 (最小 0) |

- 幅度范围 0-255，对应 DAC 输出 0 ~ ±2.83V 峰值
- 正弦波最大有效值 ≥ 2.00 Vrms

### 4.4 调制功能

SW[4:2] 组合选择调制类型：

| SW4 | SW3 | SW2 | 调制类型 | 说明 |
|-----|-----|-----|----------|------|
| 0 | 1 | 1 | **AM** 调幅 | 调制波控制载波幅度 |
| 1 | 0 | 0 | **FM** 调频 | 调制波控制载波频率 |
| 1 | 0 | 1 | **SPWM** | 正弦 PWM 调制 |
| 其他 | - | - | 无调制 | 纯净波形输出 |

调制深度调节：

| 按键 | 功能 |
|------|------|
| **BTN Right** (PB2) | 调制深度 +16 |
| **BTN Down** (PB3) | 调制深度 -16 |

### 4.5 VGA 显示内容

信号发生器模式下屏幕显示：

```
┌─────────────────────────────────┐
│                                 │
│         波形预览区域              │  ← 绿色波形实时滚动
│         (640×456)               │
│                                 │
├─────────────────────────────────┤
│ SG F:05000Hz A:200 W:N         │  ← 黄色：频率/幅度/波形
│    MOD:AM DEP:128              │  ← 青色：调制类型/深度
│    PB0/1:Amp PB2/3:Dep PB4:Trig│  ← 灰色：按键提示
└─────────────────────────────────┘
```

## 5. 示波器模式（SW3=0, SW2=0）

### 5.1 基本参数

- **通道数**: 双通道 (CH1 + CH2)
- **采样率**: ~1.5 MSPS（单通道交错采样）
- **垂直分辨率**: 8 位 (256 级)
- **水平范围**: 640 像素，对应 ~427µs 时间窗口
- **触发**: 自动触发，触发点对齐到屏幕中央

### 5.2 波形显示

| 通道 | 颜色 | 说明 |
|------|------|------|
| CH1 | **黄色** | 主通道波形（3像素宽） |
| CH2 | **蓝色** | 副通道波形（3像素宽） |
| 重叠 | **白色** | 两通道重叠区域 |

### 5.3 触发设置

| 按键 | 功能 |
|------|------|
| **BTN Left** (PB4) | 触发电平 +10（循环：10→20→...→255→10） |

### 5.4 VGA 显示内容

```
┌─────────────────────────────────┐
│  ·  ·  ·  ·  ·  ·  ·  ·       │  ← 网格 (80×54 像素)
│ ─────────────────────────       │
│     ╲ CH1 (黄色波形)             │
│       ╲                        │
│    ───╲───── CH2 (蓝色波形)      │
│         ╲                      │
│  ────────┼────────              │  ← 中心十字
│                                 │
├─────────────────────────────────┤
│ OSC DUAL                        │  ← 黄色状态栏
└─────────────────────────────────┘
```

## 6. 李萨如模式（SW3=0, SW2=1）

### 6.1 功能说明

- CH1 信号驱动 **X 轴**
- CH2 信号驱动 **Y 轴**
- 用于观察两个信号的频率比和相位差
- 显示为 **3×3 像素品红色点**

### 6.2 典型应用

| 频率比 (CH1:CH2) | 相位差 | 图形 |
|-------------------|--------|------|
| 1:1 | 0° | 对角线 |
| 1:1 | 90° | 圆形 |
| 1:2 | 0° | 8 字形 |
| 1:3 | 任意 | 三叶草 |

---

## 7. 万花筒模式（SW3=1, SW2=1）

- 将 CH1/CH2 输入信号映射为 **8 重对称**的艺术图案
- 图案中心为屏幕中央 (320, 240)
- 适合音乐可视化、艺术演示
- 每个采样点变换为 8 个对称绿色光点

## 12. 常见问题

**Q: 信号发生器无输出？**
- 检查 SW3=1, SW2=0 确认进入信号发生器模式
- 确认 DAC0832 模块连接正常
- 检查幅度设置是否过低

**Q: 示波器无波形？**
- 检查 SW3=0, SW2=0 确认进入示波器模式
- 确认 J5 扩展口连接正确（CH1: pin9-10）
- 检查输入信号幅度（XADC 输入范围 0-1V）

**Q: VGA 无显示？**
- 确认 VGA 线缆连接牢固
- 确认显示器支持 640×480@60Hz
- 检查 FPGA 是否正确配置
